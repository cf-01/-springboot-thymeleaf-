# 基于 springboot+thymeleaf 的简易果蔬管理系统


#### 介绍
【接单】基于 springboot+thymeleaf 的简易果蔬管理系统

#### 软件架构

springboot+thymeleaf+jquery+mysql+mybatis


#### 安装教程

1.  导入项目到idea
2.  修改maven设置，修改application.yml文件里数据库账号密码
3.  点击启动
